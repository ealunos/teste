O que é Variável?

Definição: Variável representa um endereço na memoria RAM; para armazenar valores de tipos diferentes exemplo(caracteres, números, simbolos, valores lógicos).

As variáveis devem ser criadas seguindo as regras

  1. Devem começar por uma letra, $ ou _.
  2. Após a primeira letra podem conter números.
  3. pela convenção, começa com letra minúscula e usa camelCase.
  4. Não pode ter espaço; use o underscore _ ex:(nota_1)
  5. JavaScript é CaseSensitive
Anotações - Lembrentes - Muito Importante !!!

Todo conteúdo de uma variável String deve estar entres (dentro) de Aspas simples ou duplas ex: let nome='Jairo';

Numeros não devem conter Aspas simples ou duplas ex: let nota1=10;
